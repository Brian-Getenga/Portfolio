About Me

Hello, I'm Brian Getenga, a passionate and results-driven software developer with a strong background in building scalable web applications and enhancing system performance. I specialize in full-stack development, utilizing technologies such as Python, Django, Bootstrap, CSS, and HTML to create efficient, user-friendly solutions. With expertise in Docker, Git version control, and general IT problem-solving, I am committed to delivering high-quality, innovative software solutions.
